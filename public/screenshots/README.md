# App Store Screenshots

This folder should contain screenshots for PWA installation.

## Required Sizes

| Size | Type | Description |
|------|------|-------------|
| 1280x720 | wide | Desktop dashboard view |
| 750x1334 | narrow | Mobile dashboard view |

## Usage

These screenshots appear in:
- Chrome's mini-infobar install prompt
- App store listings (when applicable)
- PWA installation dialogs

## Tips

1. Capture actual app screenshots at these resolutions
2. Show key features (dashboard, editor, progress)
3. Use clean, representative UI states
